# Classic Cars Collection

A beautiful, modern static webpage showcasing iconic classic American muscle cars and sports cars from the 1960s and 1970s.

## Overview

This project is a single-page website that celebrates automotive history through a curated collection of legendary classic cars. The site features a responsive, card-based design with detailed information about each vehicle.

## Features

- **Responsive Design**: Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, contemporary design with gradient backgrounds and smooth hover animations
- **Card-Based Layout**: Each car is presented in an attractive card with specifications and descriptions
- **Pure HTML/CSS**: No dependencies or build process required - just open and view

## Cars Featured

1. **1965 Ford Mustang** - The original pony car that revolutionized the industry
2. **1963 Chevrolet Corvette Sting Ray** - Iconic split-window coupe
3. **1970 Dodge Charger R/T** - Pure American muscle with HEMI power
4. **1967 Shelby GT500** - Carroll Shelby's high-performance masterpiece
5. **1969 Chevrolet Camaro Z/28** - Trans-Am racing legend
6. **1964 Pontiac GTO** - The car that started the muscle car revolution

## Technical Details

- **Technology**: HTML5, CSS3
- **Styling**: Embedded CSS with modern flexbox and grid layouts
- **Design System**: Custom gradient color scheme, card-based components
- **Typography**: System font stack for optimal performance
- **Icons**: Emoji placeholders (can be replaced with images)

## Usage

Simply open `index.html` in any modern web browser:

```bash
open index.html
```

Or double-click the file in your file explorer.

## Customization

The webpage is easy to customize:

- **Add more cars**: Copy a car card div and update the content
- **Change colors**: Modify the gradient values in the CSS
- **Add images**: Replace emoji divs with `<img>` tags pointing to your car photos
- **Update specs**: Edit the spec badges with different engine details

## Browser Compatibility
